### Inspiration

I wanted to create a macropad that would help me in my coding and 3D modeling. i needed something that will help me with copy and pasting. and i also remembered how many mistakes i do, so i had to added the undo and redo buttons! I created a macropad using 4 switches, 1 rp2040, and 2 leds. 

### Challenges

following the Fusion tutorial was a struggle. i did everything twice before making it good. but in the end the results are great.

### Specifications

BOM: 
- 4x Cherry MX Switches
- 2x SK6812 MINI Leds
- 1x XIAO RP2040
- 4x Blank DSA Keycaps
- 4x M3x16 Bolt
- 4x M3 Heatset

Others:
- KMK Firmware ( python ) 
- Top Case
- Bottom Case
